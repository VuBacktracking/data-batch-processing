### Using the project

- **Step 1. Install packages**
```bash
dbt deps
```
- **Step 2. Run dbt**

Try running the following commands:
```bash
dbt run
dbt test
```

- **Step 3 (Optional). Generate document**
```bash
dbt docs generate
dbt docs serve
```
